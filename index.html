HTML & CSS Basics Summary

HTML Summary

HTML[Hyper Text Markup Language] is used to define our page content,structure and meaning. You use it for style purpose. Use CSS instead!

  • HTML uses "elements" to describe (annotate) content

    HTML elements typically have an opening tag, content and a closing tag.

  • You can also have void[empty] elements like images.

There's a long list of available elements but you'll gain experience over time, no worries.

Learn more about all available HTML elements on
the MDN HTML element reference.
